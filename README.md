## Description

Racing Pony is a racing game where players control the car of a pony. The objective is simple, win.

## Game Controls
### Intro    
    * ENTER -> Skip Menus
    * RIGHT_SHIFT -> Insert Coin (not implemented yet)   
### Player 1 (Keyboard)
    * W -> Accelerate
    * A -> Move to the left
    * S -> Brake
    * D -> Move to the right
    * SPACE -> Turbo
## Debug Tools
    * F1 -> Check hitboxes
